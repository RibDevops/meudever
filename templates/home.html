Bem-vindo ao Sistema de Agenda

Este sistema foi desenvolvido para facilitar a visualização da agenda das aulas. O calendário mostra as aulas marcadas para cada dia, e você pode navegar entre os meses usando os botões "Mês Anterior" e "Próximo Mês".

Funcionalidades:

  • Visualização de Aulas: Veja todas as aulas agendadas organizadas por dia.
  • Agendamento de Aulas: Cadastre novas aulas clicando em “Nova Aula”.
  • Responsividade: A interface é adaptável para dispositivos móveis.
